Skipped Gyms

Omitted from the main route because they ruin momentum, have longer puzzles, more RNG, or aren't fully scouted. Beat them if you have remaining time.

Toge + Typh
If Chansey / Blissey on field: swap Typh → Exca

Rain up → swap both to Vanil + Blast
Skarmory / Moltres → swap both to Vanil + Blast
