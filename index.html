⚠️ CRITICAL: X-Item Requirements

This complete 33-gym route requires approximately 20-25 X-Items total to execute all strategies.

Item Breakdown:
• X.SpA (Special Attack) — ~16 uses ⚠️ (PRIORITY)
• X.Speed — ~3 uses
• X.Accuracy — ~2 uses
• X.Atk (Attack) — ~2 uses
• X.Special/Other — ~2 uses

⭐ Pro Tips: Stock X.SpA items FIRST (used most). Many X-Items appear in optional "If vs." branches — you can skip Driftveil, Olivine, and Blackthorn to reduce consumption.

HOENN

1. Lavaridge Town

Gym Leader: Flannery
Default Lead: Blastoise + Meloetta [Specs]

Water Spout + Helping Hand
If vs. Torkoal & Houndoom:
Water Spout + Hyper Voice
If vs. Level 85 Arcanine:
T1: X.SpA [Blastoise] + Hyper Voice
T2: Water Spout + Hyper Voice
Low chance of 4 turns. Heal if Blastoise is damaged.

2. Dewford Town

Gym Leader: Brawly
Lead: Blastoise + Vanilluxe

Water Spout + Blizzard

3. Mauville City

Gym Leader: Wattson
Lead: Typhlosion [Specs] + Excadrill

Eruption + Earthquake
If Electrode leads:
T1.1: X.Spe [Typhlosion] then T1.2: Eruption

4. Rustboro City

Gym Leader: Roxanne
Lead: Aerodactyl + Excadrill

Rock Slide + Earthquake
If Tyranitar + Probopass:
X.Speed [Excadrill] + Earthquake

5. Fortree City

Gym Leader: Winona
Lead: Aerodactyl + Vanilluxe

Rock Slide + Blizzard
If Altaria:
X.Accuracy [Aerodactyl] + Blizzard
If Pelipper:
T1: Rock Slide + Icy Wind
T2: Rock Slide + X.Atk [Aerodactyl]
T3: Rock Slide + X.Accuracy [Aerodactyl]
⬆ HEAL AFTER HOENN
